Message selector syntax here:

  http://java.sun.com/javaee/6/docs/api/javax/jms/Message.html

Required Maven version:

  This project needs Maven 2.x. 

To install dependencies:

  1) First run 

    mvn install

  2) Then follow the resulting instructions.

  3) If you are studying this example in the context of a class,
     your instructor might have given you additional information
     on downloading the missing jars.

To run:

mvn exec:java -Dexec.mainClass="edu.luc.etl.webservices.jms.stadium.Main"
